# Estruturas de Dados Básicas

Esta pasta contém simulações simples de estruturas de dados
utilizando arrays, com o objetivo de entender seu comportamento
interno e operações principais.

As implementações priorizam clareza em vez de abstrações prontas.

Este algoritmo é um ótimo exemplo para explicar o Trade-off de Memória vs. Tempo
- Tempo: Você reduziu de $O(n^2)$ para $O(n)$ (muito mais rápido!).
- Espaço: Você sacrificou um pouco de memória ($O(n)$) para armazenar o HashMap. No desenvolvimento moderno, esse é quase sempre um negócio vantajoso.

## Exercícios

- Simulação de fila (Queue)
