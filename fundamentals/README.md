# Fundamentos de Lógica

Esta pasta contém exercícios introdutórios de lógica de programação,
com foco em estruturas condicionais, laços de repetição e raciocínio sequencial.

Esses exercícios demonstram domínio dos conceitos básicos essenciais
para qualquer desenvolvedor iniciante.

## Exercícios
- FizzBuzz
- Verificação de número primo
- Cálculo de fatorial
- Sequência de Fibonacci
- Contagem de números pares e ímpares em um intervalo

## Objetivos
- Praticar controle de fluxo
- Trabalhar com condições e loops
- Desenvolver clareza de raciocínio
