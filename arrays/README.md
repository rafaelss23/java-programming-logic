# Exercícios com Arrays

Esta pasta reúne exercícios focados na manipulação de arrays,
incluindo iteração, comparação de valores e otimização de soluções.

## Exercícios
- Encontrar maior e menor valor
- Soma de elementos do array
- Remoção de elementos duplicados
- Rotação de array
- Two Sum (dois números que somam um valor alvo)

## Objetivos
- Trabalhar com índices e laços
- Resolver problemas de forma eficiente
- Comparar abordagens simples e otimizadas
