# Algoritmos Clássicos

Esta pasta apresenta algoritmos clássicos de ordenação e busca,
muito utilizados como base para avaliações técnicas.

A complexidade de tempo da Busca Binária é $O(\log n)$, onde $n$ é o número de elementos no array.

O foco está no entendimento do funcionamento e da complexidade,
não apenas na implementação.

## Exercícios
- Busca linear
- Busca binária
- Bubble Sort
- Selection Sort

## Objetivos
- Compreender algoritmos fundamentais
- Analisar complexidade de tempo
- Comparar eficiência entre abordagens
