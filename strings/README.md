# Exercícios com Strings

Esta pasta contém exercícios voltados para manipulação de strings,
explorando comparação de caracteres, iteração e lógica condicional.

Esses exercícios ajudam a desenvolver atenção a detalhes e precisão
na resolução de problemas.

## Exercícios
- Verificação de palíndromo
- Contagem de vogais
- Inversão de string sem métodos prontos
- Verificação de anagramas

## Objetivos
- Manipular texto de forma controlada
- Trabalhar com caracteres e índices
- Desenvolver soluções claras e legíveis
